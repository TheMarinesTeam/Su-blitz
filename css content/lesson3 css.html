Lesson three

  • 1-display: grid:
    Grid display: to turn an element into a grid container
    the style:

    .container{
      Display: grid;
    }

  • Grid–template-rows: property defines the number and size of the rows in a grid layout, each of the values specifies the row height.

    .contaiainer{
      Display:grid;
      grid- template- rows: 100px 300px;
    }
  • Grid–template-columns:The grid- template- columns property defines the size (width) of each columns in the grid layout.

    .contaiainer{
      Display:grid;
      grid- template- columns: 40px 150px auto 80px;
    }
  • Minimum and maximum size range:You can constrict the size range of a track by setting its minimum and maximum widths
    using the minimax0 function in place of a specific track size.

    .contaiainer{
      Display:grid;
      grid-template-columns: 200px minmax(15em , 45em) 200px;
    }
    The grid shorthand property.

    .contaiainer{
      Display: grid;
      grid: 100px 400px 100px / 200px 1fr 200px;
    }
  • 2-positioning using lines:
    The collection of these sources helps us identify the location easily,
    and we can name the grid numbering the grid lines in both directions,
    starting by setting the end of the grid column from 1 to -1. It is similar to setting it to 4.
    In our example, which contains five elements, we want the element to cover The first one in the top row is to collect the three columns.
    To do this, you use the first testicle.


    #one{
      grid-row-start: 1;
      grid-row-end: 2;
      grid-column-start: 2;
      grid-column-end: 4;
    }
    If we skip the end line declaration,
    the row will remain at its default height. Another way to achieve the result is to completely omit the end line declaration.
    We can use the (span) keyword to specify the number of paths that the elements should cover.


    #one{
      grid-column-start: span1;
      grid-column-end: span3;
    }


    There is another testicle, grid-column, grid-row.

    #one{
      grid-row: 1/ 2;
      grid-column: 1/span3;
    }
    3-Implicit grid behavior:
    The grid adjusts itself by creating more rows and columns.
    These new rows and columns are sized to fit the content by default.
    Automatic grid properties add paths as needed.


    #one{
      grid-auto-columns: 100px;
      grid-auto-rows: 100px;
    }
    Use Automatic Grid Flow to specify whether you want items to flow by row or column.


    #one{
      grid-auto-flow: dense rows;
    }


    There is another testicle, grid-column, grid-row.

    #one{
      grid-row: 1/ 2;
      grid-column: 1/span3;
    }

    shorthand.
    Rows come first, followed by columns.

    #one{
      grid-auto-flow: 12em / repeat(5, 1fr)
    }

    spacing and alignment:
    There is a gap feature that creates a space between rows and columns.

    #one{
      grid-gap: 20px 50px;
      grid-row-gap: 20px;
      grid-column-gap: 30px
    }

    When you want the grid element to fill the entire grid area, specify inline alignment with the justify-self,
    and the align-self on block.

    #one{
      justify-self: center;
      align-self: center;
    }
