Lesson four

  • 1. Spanning cells:
    Table structures can stretch cells to cover multiple rows or columns,
    use attributes like “colspan” or “rowspan”.
  • 1.1. Column Spans:
    Column spans, created with the col span attribute in the td or th element,
    stretch a cell to the right to span over the subsequent columns.
  • 1.2. Row Spans:
    Row spans, created with the row span attribute, work just like column spans,
    they cause the cell to span downward over several rows.
  • 1.3. Row Group Elements:
    In a table, you can group rows as a header, footer
    or the main body using the elements( thead , tfoot ,and tbody).
  • 1.4. Column Group Elements:
    In tables, columns are determined by the number of cells in each row.
    group columns together and add identifiers (id and class) using the colgroup element.
  • 2. Handling Overflow:
    types of overflow: visible, hidden, scroll, auto.
  • 2.1. Visible:
    which allows the content to hang out over the element box so that it all can be seen.
  • 2.2. Hidden:
    The content that does not fit gets clipped off and does not appear beyond the edges of the element’s content area.
  • 2.3. Scroll:
    scrollbars are added to the element box to let users scroll through the content.
  • 2.4. Auto:
    In most cases, scrollbars are added only when the content doesn’t fit and they are needed.
  • 3. Pseudo-classes:
    pseudo-classes in CSS let you style elements based on specific conditions, some common ones include:
  • 3.1. Hover:
    An element when the mouse hovers over it.

    .one: hover{
      background-color: blue;
    }
  • 3.2. Active:An element when it’s clicked or activated.

    .one: active{
      background-color: blue;
    }
  • 3.3. Focus Selectors:An element that’s in focus, often used for forms.

    .one:focus{
      background-color: blue;
    }
  • 3.4. Visited:Links that have been clicked, with some limitations.

    .one:visited{
      background-color: blue;
    }
  • 3.5. First-child and Last-child selectors:The first or last child element.

    li: first-child{
      border: ”solid”;
    }

    li: last-child{
      border: “solid”;
    }
  • 4. Transitions:To create one, you need to choose the property to animate and set how long the animation should last.

    one{
      transitions: 3s;
    }
  • 5. Transform:is a tool to change how something looks by turning such as:
  • 5.1. rotate():To create one, you need to choose the property to animate and set how long the animation should last.

    .a{
      transform: rotate( 20deg);
    }
  • 5.2. Scale():The scale() function scales an element by a certain factor along both the horizontal and vertical axes

    .a{
      transform: scaleY(2.4);
    }
  • 5.3. skew():The skew() function skews an element along both the horizontal and vertical axes by a certain angle.

    .a{
      transform: skewY(20deg);
    }
  • 5.4. translate():The translate() function moves an element horizontally and/or vertically on the screen.

    .a{
      transform: translate(50px, 10px);
    }
  • 6. position:controls how an element appears on a webpage. there are five position values:
  • 6.1. Static:
    Element follows the document’s normal flow.

    .one{
      position: static;
    }
  • 6.2. Relative:Element shifts from its usual spot in the flow.

    .one{
      position: relative;
      top: 40px; left:40px;
    }
  • 6.3. Absolute:The element is removed from the normal document flow ,
    and no space is created for the element in the page layout.

    .one{
      position: absolute;
      top:40px; left:40px;
    }
  • 6.4. Fixed:The element is removed from the normal document flow,
    and no space is created for the element in the page layout.

    .one{
      position: fixed;
      top: 80px;
    }
  • 6.5. sticky:The element is positioned according to the normal flow of the document,
    and then offset relative to its nearest scrolling ancestor.

    .one{
      position: sticky;
      top: 20px;
    }
  • 7. z-index property:In CSS, the z-index property is used to control the vertical stacking order of positioned elements on a web page,
    Elements with higher z-index values are displayed in front of elements with lower z-index values.

    img{
      position: absolute;
      left: 0px;
      top: 0px;
      z-index: -1;
    }
