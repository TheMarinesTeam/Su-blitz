css is crucial to a website's design capabilities.
css style are almost all visible Html elements before heading , paragraph, Link, image, lists and table.
css allow customizing attributes of the text (size, color, style, link style, image properties, list formating and table appearance).

When Linking Html to Css ,there are Two ways:
  • the first external method:
    Write an External style sheet and HTML page must include a reference to the external style sheat file
    rel="stylesheet" href=" name file.css" >
  • The internal style is defined inside the style element, inside the head section.
we have identi fied a specific class with different name . called:
  • class: Class is global attribute use with more than elemennt shared on same feature.
  • id: used to identify a single unique HTML elelment.
There are designs for CSS like:
  • background: can be either a color or images
  • background-color:is used to set a specific background to your web page
    or even a specific paragraph in the webpage:

    .one{
      Background-color:blue;
    }

  • Background-imag: we have an images and we want to added it as a background to our paragraph;
    as was mentioned before in the HTML images we have to specify the place of the image before typing the code:

    .p{
      Background-imag: url ("name")
    }
  • Padding:padding is the space bettween the content erea and the border:
    padding-top, padding-right, padding-bottom, padding-Left
    .one{
      padding: 10px, 10px, 10px, 10px;
    OR
      Padding:10px, 10px;
    }
  • Margin: has properties for specifying the margin for each side of an element
    Margin-top ,margin-right ,margin-bottom ,margin-left
    All the margin and padding properties can have the following values:

    1. auto:the browser calculates the margin
    2. length:specifies a margin in px , cm, pt, etc.
    3. %:specifies a margin in % of the width
    4. inherit: specifies that the margin should be inherit form the parent element.
    Div{
      Margin: 5px, 5px ,5px, 5px;
    OR
      Margin:5px.5px;
    }

  • Border:the css border properties allow you to specify the style, width and color of an elementâ€™s border.
  • Border style:: the border-style property can have from one to four values,
    the border style property specifies what kind of border to display:

    1. Dotted
    2. Dashed
    3. solid
    4. Double
    5. Groove value
    6. Ridge value
    7. Inset
    8. Outset
    9. None
    10. Hidden

    for Examble:

    Div{
      Border-style:solid;
    }
  • Border-color:property specifies the width of the four borders,
    the width can be set as a specific size (in px, pt,cm,em, etc) ,values :thin,thick or medium.
    Div{
      Border-color:blue;
    }
  • Border-color: property specifies the set of the color of the four borders,
    the color can be set by name or RGB values as discussed before.
    Div{
      Border-radius: 5px;
    }
  • CSS Box model: is used when talking about design and layout,
    the css box model is essentially a box that wraps around every html element;
    it consists of: borders, padding and the content

  • Content: it contains real content such as text, image, video,etc.
  • Padding: it provides padding around the content area
  • Border: it is a sub-box that surrounds the padding of the element
  • margin 5.4. Margin: this sub-box is not visible , but acts as a white space to separate two HTML elements
