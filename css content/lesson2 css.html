Lesson two

  • Flex items are arranged along flex lines. When you add “display”: flex to a div ,
    it transforms a block element to a row elements, flexbox is a powerful tool for
    creating flexible layouts on your webpage.
    the style:

    .container{
      Display:flex;
    }

  • flex-direction: Flexbox is direction-agnostic, so it doesn’t use “left”,”right”,”top”,”bottom”
    instead, we use the main axis and the cross axis, the main size is the container’s width and
    the cross size is the height, if your language reads horizontally from left to right and
    flex-direction is row, the main axis is horizontal, and the cross axis is vertical.

    .contaiainer{
      Flex-wrap:
      Flex-wrap:wrap-reverse:
    }

    .contaiainer{
      Flex-flow: wrap
      Flex-flow: wrap reverse;
    }

    Shorthand flex-direction and flex-wrap:

    .contaiainer{
      Flex-flow: row wrap;
      Flex-flow: row wrap reverse;
      OR
      Flex-flow: column wrap;
      Flex-flow: column wrap reverse;
    }
  • Flex-start and flex-end position the line of items toward the start and end of the main axis and center
    centers them, space-between, the first item is positioned at the start point, the last item goes at the end
    point, the space-around property adds an equal amount of space on the left and right side of each item.


    .contaiainer{
      Justify-conent: flex-start;
      OR
      Justify-content: flex-end;
      OR
      Justify-content: center;
      OR
      Justify-content: space-between;
      OR
      Justify-content: space-around;
    }


    When the direction is set to a column with a vertical main axis:

    .contaiainer{
      justify-content: flex-start ,flex-end ,center ,space-between , space-around;
    }
    When the direction is set to a row with a vertical main axis:


    .contaiainer{
      Justify-conent: flex-start;
      OR
      Justify-content: flex-end;
      OR
      Justify-content: center;
      OR
      Justify-content: space-between;
      OR
      Justify-content: space-around;
    }


    Flex:
    Direction==row
    Justify=> X
    Align=> Y
    Direction==column
    Justify=> Y
    Align=> X
