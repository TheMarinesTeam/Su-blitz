Spacing :

Let's learn about the shortcuts for the properties responsible for space :

  • m - for classes that set margin
  • p - for classes that set padding
  • t - for classes that set margin-top or padding-top
  • b - for classes that set margin-bottom or padding-bottom
  • s - (start) for classes that set margin-left or padding-left in LTR, margin-right or padding-right in RTL
  • e - (end) for classes that set margin-right or padding-right in LTR, margin-left or padding-left in RTL

Page Layout

What is page layout?
It is to specify the sizes and layout of the page if the web is opened on a PC or phone. It helps us organize the general appearance and arrangement of the page and prevent words and images from interfering with each other. Rather, everything has its place and remains in its shape no matter how small or large the display screen is. The size of the pictures can be reduced or the words can be placed on another line, but they do not overlap.

We already know this and used it in CSS in grid system and flexbox. So come with me, we use it in Bootstrap in a simple and better way.

2- Grid System :

Grid is made up of columns and rows that we use to plan the page. As we are accustomed to, every property has an abbreviation in Bootstrap so we can get to know it:

  • To create row in Bootstrap you will write div class="row"
  • To create column in Bootstrap you will write div class="col"
  • To make a column or row yake a more than size , you can write : div class="row-3" , div class="col-3" .
    you can change the number to you need.
  • You can control the display method according to the screen size :
    Extra small (xs) , Small (sm) , Medium (md) , Large (lg) , Extra large (xl) and Extra extra large (xxl).
