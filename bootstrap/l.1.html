What is Bootstrap ?

Bootstrap is a framework that contains a ready-made CSS file that we use in your project.
It contains a large collection of classes with CSS effects.
To use these effects, we go and assign the element a class.
Our job is to know how and when to use them. What do we benefit from bootstrap? It saves a lot of time and effort.
The disadvantage of Bootstrap is that you cannot rely on it completely because not every framework performs everything.

We will use Bootstrap 5 , because it is the latest version .

There are conditions to use Bootstrap
1- you must have learned CSS , JS and HTML . if you don`t learned them you can learned from our website.
2- we need to files Bootstrap , css , js and webfonts .

Before we continue, I ask you to open the official Bootstrap website and take a look at the utilities available for each property, because Bootstrap depends on writing the utilities in Classes.

There are utilities in Bootstrap, which is an abbreviation for property, and we write it in the class. such as : bg = background-color , m = margin , d = display. ......etc.

Classes control the design of the appearance of elements, such as: (button, nav and table).
If I told you to make the background red You will go to the css file , call Body, and write the property responsible for creating this command.
This is true, but with Bootstrap you can do this in a more concise way and write the css properties Enter the Classes in a short and simple way.

There are utilities for every property.
All you have to do is go to the Bootstrap website and search for utilities and you will find more than one type. Take a look at them before we continue.

4- Opacity

The opacity property sets the opacity level for an element. The opacity level describes the transparency level, where 1 is not transparent at all, .5 is 50% visible, and 0 is completely transparent.
